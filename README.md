# 💰 Taxifyy - Фінансовий калькулятор для ФОП

Комплексний фінансовий інструмент для підприємців (ФОП 3 групи) та фрілансерів з підтримкою мультивалютності та автоматичним розрахунком податків, доходів та комунальних послуг.

[![Live Demo](https://img.shields.io/badge/demo-live-green?style=for-the-badge)](https://taxes-calculator-delta.vercel.app/)
![License](https://img.shields.io/badge/license-MIT-blue?style=for-the-badge)

---

## ✨ Можливості

### 💼 **Розрахунок доходу**
- 📅 **Інтерактивний календар** для вибору відпрацьованих днів
- 💵 **Автоматичне підвантаження курсів валют** з НБУ (USD, EUR, PLN)
- 💸 Підтримка **власних курсів** та **рейтів за годину**
- 🧾 Детальний розрахунок **чистого доходу** після всіх податків
- 🔢 Автоматичний підрахунок робочих днів та годин

### 🧾 **Податковий калькулятор**
- 📊 **Розрахунок податків за місяць або квартал**
- 🧾 Розрахунок:
  - **Єдиного податку (5%)**
  - **Військового збору (1%)**
  - **ЄСВ (22% від мінімальної ЗП)**
  - **Чистого доходу**
- 📆 Розрахунок **окремо по місяцях та по кварталах**

### 💱 **Обмін валют та статистика**
- 📈 **Графіки курсів валют** за період
- 🔄 **Конвертер валют** в реальному часі
- 📊 **Статистика** курсів за тиждень/місяць
- 💱 Підтримка **USD, EUR, PLN**

### 🏠 **Калькулятор комуналки**
- 🏠 Розрахунок **води, опалення, електрики**
- 📊 **Лічильники** з попередніми та поточними показниками
- 💰 Автоматичний розрахунок **за тарифами**
- ➕ **Додаткові платежі** (інтернет, сміття тощо)
- 🔄 **Перетягування карток** для зміни порядку (desktop)

### ⚙️ **Налаштування**
- 💱 **Мультивалютність** (USD, EUR, PLN)
- 🔢 **Точність чисел** (0-4 десяткових знаки)
- 🎨 **Тема оформлення** (світла/темна/авто)
- 📋 **Показ/приховування** деталізації податків
- 💾 **Автозбереження** всіх налаштувань

### 🌐 **Навігація**
- 🔗 **URL маршрутизація** (/income, /taxes, /exchange, /utilities)
- 📱 **Адаптивний дизайн** для всіх пристроїв
- 💾 **Локальне збереження** даних
- 🇺🇦 Повна підтримка **української мови**

## 🧮 Як використовувати

### 1. **Розрахунок доходу** (`/income`)
Введіть години роботи та рейт за годину. Калькулятор автоматично:
- Підвантажить актуальний курс валюти з НБУ
- Розрахує брудний дохід
- Вирахує всі податки (5% єдиний + 1% військовий + 22% ЄСВ)
- Покаже чистий дохід

### 2. **Податковий калькулятор** (`/taxes`)
Введіть ваш дохід та виберіть період (місяць/квартал):
- Розрахунок всіх податків окремо
- Можливість квартального планування
- Детальна розбивка по видах податків

### 3. **Обмін валют** (`/exchange`)
- Перегляд актуальних курсів
- Конвертація між валютами
- Графіки зміни курсів за період

### 4. **Комуналка** (`/utilities`)
- Введіть показники лічильників
- Додайте тарифи
- Додаткові платежі
- Перетягуйте картки для зміни порядку

### 5. **Налаштування**
Налаштуйте валюту, точність чисел, тему та інші параметри.

## 🛠️ Розробка

### 🔧 Використані технології

- **React.js** + **React Router** - навігація та компоненти
- **CSS Modules** - модульні стилі
- **Vite** - збірка та розробка
- **Vercel** - хостинг і CI/CD
- **NBU API** - актуальні курси валют
- **Chart.js** - графіки та візуалізація
- **React Icons** - іконки інтерфейсу

### 📦 Встановлення

```bash
git clone https://github.com/qboww/taxifyy.git
cd taxifyy
npm install
npm run dev
```

### 🚀 Збірка для продакшену

```bash
npm run build
npm run preview
```
